Спасибо за интерес к проекту **EC-Arithmetic-and-logical-foundations-of-computing-technology**!

* Материалы должны быть:
  * понятными
  * структурированными

* Избегайте избыточной сложности
* Соблюдайте единый стиль оформления
* Проверяйте орфографию и форматирование

## 🔧 Как внести вклад

1. Сделайте Fork репозитория
2. Создайте новую ветку:

   ```bash
   git checkout -b feature/your-change
   ```

3. Внесите изменения
4. Сделайте commit:

   ```bash
   git commit -m "Добавлена тема: Работа с ..."
   ```

5. Отправьте изменения:

   ```bash
   git push origin feature/your-change
   ```

6. Создайте Pull Request

## ✅ Перед отправкой Pull Request

Проверьте:

* [ ] Нет опечаток
* [ ] Markdown корректно отображается
* [ ] Файлы лежат в нужных папках
* [ ] Названия файлов логичны
* [ ] Нет лишних или временных файлов

## 🚫 Чего делать не стоит

* Не ломать существующую структуру
* Не добавлять нерелевантный контент
* Не использовать разные стили оформления в одном разделе
* Не загружать большие файлы без необходимости

## 💬 Обсуждение

Если вы не уверены, как правильно оформить материал — создайте Issue и опишите идею.
