# Лабораторная работа № 4  
Применение компьютерных программ для анализа текущего состояния устройства и аппаратного обеспечения  

## Цель и задачи  

1. Ознакомиться с методами сбора информации о состоянии ПК с помощью Python‑скриптов.  
2. Научиться использовать библиотеки `psutil`, `platform`, `subprocess` для получения системных данных.  
3. Научиться анализировать и визуализировать собранные данные в виде таблицы и графика.  
4. Разработать простое приложение, выводящее сведения о CPU, памяти, дисках, сети и подключённых устройствах.  

## Теоретическая часть  

| Понятие | Определение | Пример/Формула |
|---|---|---|
| **Процесс** | Независимая исполняемая сущность, имеющая собственный адресный пространство. | `psutil.Process(pid)` |
| **CPU‑загрузка** | Процент времени процессора, занятого процессом. | `psutil.cpu_percent(interval=1)` |
| **Оперативная память (RAM)** | Физическая память компьютера, доступная для выполнения программ. | `psutil.virtual_memory()` |
| **Swap‑память** | Виртуальная память на диске, используемая при нехватке RAM. | `psutil.swap_memory()` |
| **Диск** | Устройство хранения данных, имеющее файловую систему. | `psutil.disk_partitions()` |
| **Сеть** | Механизм передачи данных между компьютерами. | `psutil.net_io_counters()` |
| **USB‑устройства** | Переносные устройства, подключаемые через порт USB. | `subprocess.check_output(['lsusb'])` |
| **Библиотека `psutil`** | Платформенно‑независимый модуль для получения информации о системе. | `pip install psutil` |
| **Библиотека `platform`** | Модуль стандартной библиотеки Python для получения информации о платформе. | `platform.uname()` |
| **Библиотека `subprocess`** | Модуль для запуска внешних команд. | `subprocess.run(['ipconfig', '/all'], capture_output=True)` |

## Ход выполнения работы  

1. **Инициализация проекта**  
   - Создайте папку `lab4` и файл `main.py`.  
   - Установите необходимые библиотеки:  
     ```bash
     pip install psutil
     ```

2. **Получение общей информации о системе**  
   - Используйте `platform.uname()` для вывода имени ОС, версии и архитектуры.  
   ```python
   import platform
   def print_system_info():
       info = platform.uname()
       print(f"System: {info.system}")
       print(f"Node name: {info.node}")
       print(f"Release: {info.release}")
       print(f"Version: {info.version}")
       print(f"Machine: {info.machine}")
       print(f"Processor: {info.processor}")
   ```

3. **Анализ процессора**  
   - С помощью `psutil.cpu_percent()` получите текущую загрузку CPU.  
   - Выведите количество ядер.  
   ```python
   import psutil
   def cpu_stats():
       print(f"CPU cores: {psutil.cpu_count(logical=True)}")
       print(f"CPU usage: {psutil.cpu_percent(interval=1)}%")
   ```

4. **Память и swap**  
   - Получите объёмы RAM и swap.  
   ```python
   def memory_stats():
       vm = psutil.virtual_memory()
       sm = psutil.swap_memory()
       print(f"Total RAM: {vm.total / (1024**3):.2f} GB")
       print(f"Used RAM: {vm.used / (1024**3):.2f} GB ({vm.percent}%)")
       print(f"Total Swap: {sm.total / (1024**3):.2f} GB")
       print(f"Used Swap: {sm.used / (1024**3):.2f} GB ({sm.percent}%)")
   ```

5. **Диски**  
   - Выведите список разделов и их свободное место.  
   ```python
   def disk_stats():
       partitions = psutil.disk_partitions()
       for p in partitions:
           usage = psutil.disk_usage(p.mountpoint)
           print(f"Mount point: {p.mountpoint}")
           print(f"  Total: {usage.total / (1024**3):.2f} GB")
           print(f"  Used: {usage.used / (1024**3):.2f} GB ({usage.percent}%)")
   ```

6. **Сетевые интерфейсы**  
   - Получите статистику ввода/вывода.  
   ```python
   def network_stats():
       net = psutil.net_io_counters()
       print(f"Bytes sent: {net.bytes_sent / (1024**2):.2f} MB")
       print(f"Bytes received: {net.bytes_recv / (1024**2):.2f} MB")
   ```

7. **USB‑устройства (только для Windows и Linux)**  
   - Запускайте внешнюю команду `wmic` или `lsusb`.  
   ```python
   import subprocess
   def usb_devices():
       try:
           output = subprocess.check_output(['lsusb']).decode()
           print("USB devices:\n" + output)
       except Exception as e:
           print(f"USB enumeration not supported: {e}")
   ```

8. **Главная функция**  
   ```python
   def main():
       print_system_info()
       print()
       cpu_stats()
       print()
       memory_stats()
       print()
       disk_stats()
       print()
       network_stats()
       print()
       usb_devices()

   if __name__ == "__main__":
       main()
   ```

9. **Запуск и проверка**  
   - Выполните скрипт `python main.py`.  
   - Сравните вывод с реальным состоянием системы.
