# Практическая работа № 1  
Международная система единиц для описания характеристик устройств

## Цель и задачи
1. Ознакомиться с основными единицами Международной системы единиц (SI) и их применением в технической документации.  
2. Научиться преобразовывать величины из одной системы измерений в SI, используя префиксы и коэффициенты.  
3. Разработать простое Python‑приложение, которое принимает пользовательские данные и выводит их в SI.  
4. Оформить отчёт, анализирующий результаты работы и выводы о применении SI в инженерных задачах.

## Теоретическая часть
1. **Международная система единиц (SI)** – система, основанная на семи базовых величинах: длина (метр, m), масса (килограмм, kg), время (секунда, s), сила тока (ампер, A), температура (Кельвин, K), количество вещества (моль, mol), световой поток (кандела, cd).  
2. **Префиксы SI** – множители, обозначающие порядок величины: кило‑ (10³), мега‑ (10⁶), гигга‑ (10⁹), деци‑ (10⁻¹), центi‑ (10⁻²), мили‑ (10⁻³), микро‑ (10⁻⁶), нано‑ (10⁻⁹).  
3. **Единицы производных величин** – получаются умножением базовых единиц, напр. скорость \(v = \frac{s}{t}\) измеряется в м/с (м/с).  
4. **Конвертация** – для перехода из одной единицы в другую умножаем на коэффициент преобразования:  
   \[
   1\,\text{km} = 1000\,\text{m}, \quad 1\,\text{kg} = 1000\,\text{g}
   \]  
5. **Коэффициент перехода** – в Python можно хранить в словаре:  
   ```python
   conv = {"km": 1000, "g": 0.001}
   ```
6. **Формулы SI** – пример: мощность \(P = \frac{W}{t}\) измеряется в ваттах (Вт).  
7. **Стандарты** – ГОСТ 2.102‑2016 «Международная система единиц» определяет правила записи.  
8. **Кратность** – при работе с большими/маленькими величинами удобно использовать префиксы: 1 kW = 1000 W.  
9. **Преобразование дат** – для временных меток можно использовать `datetime.timedelta` и перевод в секунды.  
10. **Проверка единиц** – в Python библиотека `pint` позволяет управлять единицами, но в данной работе будем использовать простые словари.

## Ход выполнения работы
1. **Подготовка окружения**  
   - Открыть IDE (PyCharm, VSCode) и создать новый проект `si_converter`.  
   - Создать файл `converter.py`.

2. **Определение словаря префиксов**  
   ```python
   PREFIXES = {
       "k": 1e3,
       "M": 1e6,
       "G": 1e9,
       "m": 1e-3,
       "μ": 1e-6,
       "n": 1e-9,
   }
   ```

3. **Функция распознавания единицы**  
   ```python
   def parse_value(unit_str: str) -> tuple[float, str]:
       """Разбирает строку вида '5kW' и возвращает (5.0, 'W')."""
       import re
       match = re.fullmatch(r'(\d+(\.\d+)?)([a-zA-Zμ]+)', unit_str)
       if not match:
           raise ValueError("Неверный формат")
       value, _, unit = match.groups()
       return float(value), unit
   ```

4. **Функция преобразования в SI**  
   ```python
   def to_si(value: float, unit: str) -> tuple[float, str]:
       """Переводит значение в базовую SI‑единицу."""
       for pref, mult in PREFIXES.items():
           if unit.startswith(pref):
               base_unit = unit[len(pref):]
               return value * mult, base_unit
       return value, unit
   ```

5. **Главная логика программы**  
   ```python
   def main():
       inp = input("Введите величину (например, 5kW): ")
       val, unit = parse_value(inp)
       si_val, si_unit = to_si(val, unit)
       print(f"{si_val} {si_unit} (в SI)")
   
   if __name__ == "__main__":
       main()
   ```

6. **Тестирование**  
   - Ввести `3.5kW` → `3500.0 W`.  
   - Ввести `1200m` → `1200.0 m`.  
   - Ввести `2.5μs` → `2.5e-06 s`.

7. **Документирование**  
   - Добавить комментарии к функциям.  
   - Создать файл `README.md` с описанием работы.

8. **Формирование отчёта**
