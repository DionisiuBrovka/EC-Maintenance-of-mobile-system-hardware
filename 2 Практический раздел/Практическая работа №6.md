# Практическая работа № 6  
Схемы контроля передачи информации из регистра в регистр при помощи кода проверки на четность  

## Цель и задачи  
1. Изучить принципы кодирования данных с использованием кода проверки на четность.  
2. Научиться проектировать и реализовывать схемы передачи данных между регистрами с контролем целостности.  
3. Разработать программу на Python, моделирующую передачу и проверку данных.  
4. Оценить влияние ошибок на передачу и способы их обнаружения.  

## Теоретическая часть  
| Понятие | Определение | Пример/Формула |
|---------|-------------|----------------|
| Регистры | Оперативные ячейки памяти, хранящие данные в цифровой форме | `R1`, `R2` |
| Код проверки на четность | Дополнительный бит, определяющий, чётное ли число единиц в коде | `P = (Σ bits) mod 2` |
| Чётный код | Код, в котором суммарное число единиц чётно | `0101` → 2 единицы → чётный |
| Нечётный код | Код, в котором суммарное число единиц нечётно | `0111` → 3 единицы → нечётный |
| Ошибка передачи | Изменение одного или более битов при передаче | `0101 → 1101` (первый бит изменён) |
| Схема контроля | Механизм проверки целостности данных при приёме | Чётность + XOR‑операция |
| XOR‑операция | Побитовое исключающее ИЛИ | `1 XOR 0 = 1`, `1 XOR 1 = 0` |
| Паритетный бит | Бит, добавляемый к данным для обеспечения контроля | `data + parity` |
| Проверка ошибок | Процесс сравнения полученного кода с ожидаемым | Если `parity` не совпадает → ошибка |
| Корректировка ошибок | Восстановление данных при наличии корректирующего кода | В данной работе только обнаружение |

## Ход выполнения работы  

1. **Подготовка среды**  
   - Установить Python 3.10+ (если ещё не установлен).  
   - Создать проект `parity_transfer` и файл `parity_transfer.py`.  

2. **Функция расчёта паритета**  
   ```python
   def calculate_parity(data: int, bits: int = 8) -> int:
       """Возвращает паритетный бит (чётный) для заданного значения."""
       parity = 0
       for i in range(bits):
           parity ^= (data >> i) & 1
       return parity
   ```  
   *Пояснение:* используем XOR‑операцию для подсчёта количества единиц.  

3. **Формирование пакета для передачи**  
   ```python
   def pack_data(data: int, bits: int = 8) -> int:
       """Упаковывает данные с добавлением паритетного бита."""
       parity = calculate_parity(data, bits)
       return (data << 1) | parity
   ```  
   *Пример:* `data = 0b10101010` → `0b101010101` (9‑битный пакет).  

4. **Симуляция передачи с возможной ошибкой**  
   ```python
   import random

   def transmit(packet: int, error_rate: float = 0.1) -> int:
       """Передаёт пакет с вероятностью ошибки битов."""
       for i in range(packet.bit_length()):
           if random.random() < error_rate:
               packet ^= 1 << i  # инвертируем бит
       return packet
   ```  
   *Пояснение:* `error_rate` задаёт вероятность изменения любого бита.  

5. **Проверка полученного пакета**  
   ```python
   def verify_packet(packet: int, bits: int = 8) -> bool:
       """Проверяет корректность пакета по паритету."""
       data = packet >> 1
       received_parity = packet & 1
       expected_parity = calculate_parity(data, bits)
       return received_parity == expected_parity
   ```  

6. **Главная функция и демонстрация**  
   ```python
   def main():
       data = 0b11001100  # пример данных
       packet = pack_data(data)
       received = transmit(packet)
       if verify_packet(received):
           print("Передача успешна, данные:", bin(received >> 1))
       else:
           print("Ошибка передачи!")

   if __name__ == "__main__":
       main()
   ```  
   *Результат:* При низком `error_rate` почти всегда «Передача успешна».
