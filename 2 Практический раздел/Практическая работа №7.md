4. Отразить результаты в письменном отчёте, продемонстрировав умение аргументированно оценивать корректность работы программы.

## Теоретическая часть
| № | Понятие | Краткое описание |

## Ход выполнения работы

1. **Подготовка среды**  
   - Установить Python 3.12 (если ещё не установлен).  
   - Создать виртуальное окружение и установить библиотеки: `pandas`, `numpy`, `matplotlib`, `pyyaml`.  
   ```bash
   python -m venv venv
   source venv/bin/activate   # Windows: venv\Scripts\activate
   pip install pandas numpy matplotlib pyyaml
   ```

2. **Создание шаблона конфигурации**  
   - Создать файл `config.yaml` с пороговыми значениями и путями к данным.  
   ```yaml
   data_path: "./data/measurements.csv"
   thresholds:
     temperature: 75.0
     voltage: 3.3
   output:
     report: "./reports/report.txt"
     plot: "./reports/plot.png"
   ```

3. **Чтение и первичная обработка данных**  
   - Считать CSV, проверить типы столбцов, заполнить пропуски средним значением.  
   ```python
   import pandas as pd
   import yaml

   with open('config.yaml') as f:
       cfg = yaml.safe_load(f)

   df = pd.read_csv(cfg['data_path'])
   df = df.fillna(df.mean())
   ```

4. **Фильтрация выбросов**  
   - Применить Z‑score для удаления экстремальных значений.  
   ```python
   from scipy import stats
   df = df[(np.abs(stats.zscore(df)) < 3).all(axis=1)]
   ```

5. **Диагностика**  
   - Сравнить каждое измерение с порогами, пометить как `OK` или `FAIL`.  
   ```python
   def check(row):
       return all(row[metric] <= cfg['thresholds'][metric]
                  for metric in cfg['thresholds'])
   df['status'] = df.apply(check, axis=1)
   ```

6. **Визуализация**  
   - Построить график температур и напряжений с цветовой кодировкой по статусу.  
   ```python
   import matplotlib.pyplot as plt

   colors = df['status'].map({'True': 'green', 'False': 'red'})
   plt.scatter(df.index, df['temperature'], c=colors, label='Temperature')
   plt.scatter(df.index, df['voltage'], c=colors, label='Voltage')
   plt.legend()
   plt.savefig(cfg['output']['plot'])
   ```

7. **Составление отчёта**  
   - Сгенерировать текстовый файл с резюме: общее количество измерений, число ошибок, средние значения.  
   ```python
   with open(cfg['output']['report'], 'w') as f:
       f.write(f"Total samples: {len(df)}\n")
       f.write(f"Failed samples: {df['status'].value_counts()[False]}\n")
   ```

8. **Логирование**  
   - Добавить запись начала и конца выполнения, ошибки при чтении.  
   ```python
   import logging
   logging.basicConfig(filename='diagnostic.log', level=logging.INFO)
   logging.info('Diagnostic run started')
   # ... код ...
   logging.info('Diagnostic run finished')
   ```

9. **Тестирование**  
   - Написать простые unit‑тесты для функций `check` и `filter`.  
   ```python
   import unittest

   class TestDiagnostics(unittest.TestCase):
       def test_check(self):
           row = pd.Series({'temperature': 70, 'voltage': 3.1})
           self.assertTrue(check(row))
   ```

10. **Подготовка к сдаче**  
    - Упаковать исходники, отчёт, график в архив `submission.zip`.  
    - Включить README с инструкциями по запуску.
