# Технические характеристики мобильных устройств. Техническое обслуживание мобильных устройств

## Цели и задачи лекции

## Ключевые понятия
- **Процессор (CPU)** – центральный вычислительный узел, отвечающий за выполнение инструкций.  
- **GPU** – графический процессор, ускоряет обработку визуальных данных.  
- **RAM** – оперативная память, временное хранение данных в процессе работы.  
- **ROM** – неизменяемая память, хранящая прошивку и системные файлы.  
- **Аккумулятор (Battery)** – источник питания, измеряется в мАч (миллАч).  
- **Калибровка** – корректировка параметров сенсоров (экран, гироскоп, акселерометр).  
- **OTA‑обновление** – обновление прошивки по воздуху.  
- **Диагностический режим** – режим, в котором устройство предоставляет подробную информацию о состоянии компонентов.  
- **Тесты на износ** – процедуры, измеряющие износ аккумулятора, памяти и других узлов.  
- **Системный лог** – запись всех событий и ошибок, используемая для диагностики.

### 3. Техническое обслуживание (35 мин)
- **Обновление прошивки (OTA)**  
  - Процесс, риски, rollback  
- **Диагностика и калибровка**  
  - Использование ADB, Android Debug Bridge, iOS Diagnostics  
- **Тесты на износ аккумулятора**  
  - Измерение ёмкости, коэффициент деградации  
- **Чистка и замена компонентов**  
  - Как безопасно открывать корпус, заменять аккумулятор, дисплей  

```python
# Пример Python‑скрипта для чтения температуры CPU на Android через ADB
import subprocess

def get_cpu_temp():
    cmd = ["adb", "shell", "cat", "/sys/class/thermal/thermal_zone0/temp"]
    temp = subprocess.check_output(cmd).decode().strip()
    return int(temp) / 1000  # в °C

print(f"Температура CPU: {get_cpu_temp()} °C")
```
