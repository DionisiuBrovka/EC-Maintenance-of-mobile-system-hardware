# Процессы, происходящие при включении электропитания

## Цели и задачи лекции
- Понять последовательность событий, происходящих при включении электропитания устройства.
- Изучить роль компонентов: источника питания, разрядных устройств, схемы защиты.
- Научиться анализировать схемы включения и выявлять потенциальные проблемы.
- Рассмотреть примеры реализации в микроконтроллерах и системах на Python.

## Ключевые понятия
- **Электропитание** – источник электрической энергии, обеспечивающий работу устройства.
- **Разрядный резистор (pull‑down)** – элемент, удерживающий вход в низком состоянии при отсутствии сигнала.
- **Триггер** – логический элемент, сохраняющий состояние до изменения входа.
- **Магнитный клапан** – устройство, создающее задержку в цепи при включении.
- **Питание от аккумулятора** – источник, обеспечивающий автономную работу.
- **Периферийные устройства** – компоненты, подключенные к контроллеру (датчики, дисплеи).
- **Кэш** – память, содержащая данные для быстрого доступа после включения.
- **Инициализация** – процесс настройки устройств перед использованием.
- **Задержка (delay)** – пауза, необходимая для стабилизации сигналов.
- **Проверка целостности** – тестирование правильности работы цепей после включения.

### 3. Логика инициализации микроконтроллера (20 мин)
- Секвенция старта: RESET → PLL → Clock → Peripherals.  
- Пример кода инициализации в Python с использованием MicroPython.

```python
import machine
import time

# Инициализация таймера
tim = machine.Timer(-1)
tim.init(period=1000, mode=machine.Timer.PERIODIC, callback=lambda t: print('Tick'))

# Инициализация порта
led = machine.Pin(2, machine.Pin.OUT)
led.value(1)

time.sleep(1)
led.value(0)
```

### 5. Практические примеры на Python (10 мин)
- Считывание датчика после включения.  
- Периферийные устройства: OLED, SPI, I2C.  

```python
import machine
import ssd1306

i2c = machine.I2C(-1, scl=machine.Pin(5), sda=machine.Pin(4))
display = ssd1306.SSD1306_I2C(128, 64, i2c)

display.text('Hello', 0, 0)
display.show()
```
