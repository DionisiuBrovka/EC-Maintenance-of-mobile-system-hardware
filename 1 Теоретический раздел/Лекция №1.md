## Основное содержание

### 1. Введение (5 мин)
- Что такое учебный предмет и почему его целеполагание важно.
- Краткое описание предмета «Инструментальное программное обеспечение» и его роли в образовательной программе.

### 2. Цели учебного предмета (15 мин)
- Формирование базовых знаний о языках программирования и IDE.
- Развитие навыков работы с системами контроля версий и CI/CD.
- Подготовка к практической работе в командах разработки.
- Формирование умений критически оценивать и выбирать инструменты.

### 3. Задачи предмета (15 мин)
- Изучение архитектуры программных продуктов и их компонентов.
- Практическая работа с популярными IDE (PyCharm, VS Code).
- Настройка окружения разработки (Docker, виртуальные машины).
- Работа с системами сборки и тестирования (pytest, tox).
- Создание и поддержка документации (Sphinx, MkDocs).

#### 4.2 Системное администрирование (5 мин)

#### 4.4 Управление проектами (5 мин)

```yaml
# Пример файла GitHub Actions для CI
name: CI

on:
  push:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Set up Python
        uses: actions/setup-python@v2
        with:
          python-version: '3.9'
      - name: Install dependencies
        run: pip install -r requirements.txt
      - name: Run tests
        run: pytest
```
