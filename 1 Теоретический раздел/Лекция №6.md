## Цели и задачи лекции

- Применить полученные знания на практических примерах с использованием Python.

## Ключевые понятия  
- **Цифровая схема** – электронная схема, работающая с дискретными уровнями напряжения.  
- **Диагностика** – процесс выявления неисправностей в цифровой схеме.  
- **Тестовой стенд** – аппаратно‑программный комплекс для проверки схем.  
- **Логический анализатор** – устройство для записи многоканальных логических сигналов.  
- **Временная база** – точное измерение времени событий.  
- **JTAG** – протокол отладки и программирования микросхем.  
- **Кросс‑сборка** – сборка кода для целевого микроконтроллера на другой платформе.  
- **Профилирование** – измерение производительности цифровой системы.  
- **Алгоритм тестирования** – последовательность действий для проверки корректности схемы.  
- **Периферийные интерфейсы** – внешние средства ввода/вывода схемы.

- **Логические анализаторы** – принципы работы, выбор частоты дискретизации.

- **Программируемые логические контроллеры** – использование в тестовых стендах.  
- **Сценарий тестирования** – пример скрипта на Python для управления логическим анализатором.

```python
import pyvisa
rm = pyvisa.ResourceManager()
analyzer = rm.open_resource('USB0::0x1234::0x5678::INSTR')
analyzer.write('RUN')
data = analyzer.read_bytes()
print(len(data))
```

### 3. Программное обеспечение для анализа сигналов (20 мин)  
- Библиотеки: PyVisa, PyVISA‑Py, NumPy, Matplotlib.  
- Пример обработки логических данных: фильтрация, коррекция задержек.  
- Создание собственных профилей тестов с помощью Python.  

```python
import numpy as np
import matplotlib.pyplot as plt

# Считываем логические данные
signals = np.frombuffer(data, dtype=np.uint8)

# Фильтрация шума
filtered = np.where(signals > 0, 1, 0)

plt.step(filtered)
plt.title('Фильтрованные сигналы')
plt.xlabel('Время (с)')
plt.ylabel('Уровень')
plt.show()
```
