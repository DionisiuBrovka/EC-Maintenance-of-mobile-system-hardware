# Методы контроля передачи информации. Методы контроля арифметических и логических операций

## Цель и задачи занятия  
1. Понять, зачем нужны проверки корректности данных.  
2. Познакомиться с основными методами контроля: CRC, Parity, Checksum, Digital Signature.  
3. Научиться применять простые проверки в Python‑программах.  
4. Развить уверенность в том, что «проверка» – это не дополнительная работа, а защита от ошибок.

## Почему это важно?  
В реальной жизни данные перемещаются по сети, сохраняются на дисках и передаются по каналам связи. Ошибки могут возникать из‑за шумов, сбоев оборудования, человеческих ошибок. Без контроля вы получите «потерянный» файл, неверный результат вычислений или даже уязвимость к взлому. Контроль делает ваш код надёжным и безопасным.

## Основные понятия и идеи  
| Термин | Что это? | Краткая аналогия |
|--------|----------|-------------------|
| **Parity (чётность)** | Один бит, показывающий, сколько единиц в наборе. | Как «проверка» в игре «угадай число» – всегда можно проверить, что число чётное. |
| **Checksum (контрольная сумма)** | Сумма всех байтов (или слов), иногда с инверсиями. | Счёт в игре «Пинг‑Понг» – сумма очков должна совпадать. |
| **CRC (циклическая избыточность)** | Алгоритм, использующий полиномы, выдаёт «отпечаток» блока данных. | Печатный отпечаток на ладони – уникален и позволяет быстро проверить подлинность. |
| **Digital Signature (цифровая подпись)** | Хэш + шифрование приватным ключом, подтверждающий авторство. | Подпись почты – гарантирует, что письмо от того, кто он есть. |
| **Hash (хэш)** | Фиксированная длина, получаемая из произвольных данных. | Короткая «отметка» книги – один и тот же текст всегда даёт один и тот же хэш. |
| **Integrity (целостность)** | Гарантия, что данные не изменились. | Проверка, что посылка не была открыта в пути. |
| **Error‑Detection (обнаружение ошибок)** | Способ найти ошибку, но не всегда исправить её. | Проверка, что билет в кино действительно существует. |

## Объяснение темы

### [10 мин] Вводная: почему нужна проверка?  
- Быстрый пример: передача сообщения “HELLO” по сети может стать “HELOO” из‑за шумов.  
- Без контроля вы не заметите, что получили неправильные данные.  
- С помощью простого «проверочного бита» можно сразу узнать, что что‑то не так.

### [10 мин] Parity – самый простой метод  
```python
def parity_bit(data: bytes) -> int:
    """Возвращает 0, если количество единиц чётно, иначе 1."""
    return sum(bin(b).count('1') for b in data) % 2

# Пример
msg = b"test"
print(parity_bit(msg))  # 0
```
- Мини‑задание: посчитайте parity для строки `"abc"`.

### [10 мин] Checksum – сумма с инверсией  
```python
def simple_checksum(data: bytes) -> int:
    """Сумма всех байтов, взятая по модулю 256."""
    return sum(data) % 256

msg = b"hello"
print(simple_checksum(msg))  # 532 % 256 = 20
```
- Мини‑задание: как изменится checksum, если поменять один символ?  
  *Подсказка:* посмотрите, как меняется сумма.

### [10 мин] CRC – более надёжный отпечаток  
- В Python есть библиотека `crcmod`.  
- Пример использования CRC‑32:  
```python
import zlib
data = b"important data"
crc = zlib.crc32(data)
print(f"CRC‑32: {crc:#010x}")
```
- Мини‑задание: вычислите CRC‑16 для строки `"123456789"` (см. таблицу полиномов в ссылках).

### [5 мин] Digital Signature – защита от подделки  
- Ключи RSA, простая подпись.  
- Пример с `cryptography`:
```python
from cryptography.hazmat.primitives import hashes, serialization
from cryptography.hazmat.primitives.asymmetric import padding, rsa

private_key = rsa.generate_private_key(public_exponent=65537, key_size=2048)
message = b"Secure message"

signature = private_key.sign(
    message,
    padding.PSS(mgf=padding.MGF1(hashes.SHA256()), salt_length=padding.PSS.MAX_LENGTH),
    hashes.SHA256()
)
```
- Мини‑задание: попробуйте проверить подпись с помощью публичного ключа.

## Итог и выводы  
- Проверки данных – это фундамент надёжной коммуникации.  
- Parity и Checksum подходят для простых ошибок, CRC – для более сложных.  
- Digital Signatures добавляют авторизацию и защищают от подделки.  
- В Python реализация всех этих методов проста и доступна через стандартные библиотеки.
