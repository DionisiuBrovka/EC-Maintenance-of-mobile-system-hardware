# Практическая работа № 3: Работа с технической документацией для схемного контроля

## Цель занятия  
1. Научить студентов эффективно извлекать и использовать сведения из технической документации при разработке схемного контроля.  
2. Развить навыки работы с инструментами Python для анализа и визуализации технических данных.

## Задачи занятия  
- Ознакомить с форматом и содержанием типовой технической документации (спецификации, схемы, таблицы).  
- Научить использовать библиотеки Pandas, NumPy и Matplotlib для обработки данных из таблиц.  
- Реализовать простую программу, генерирующую графики схемного контроля на основе загруженных данных.  
- Оценить корректность полученных результатов по заданным критериям.  

## План занятия (90 мин)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала  
  - Введение в структуру технической документации — 10 мин  
  - Работа с таблицами в Python (Pandas) — 15 мин  
  - Визуализация схемного контроля (Matplotlib) — 10 мин  
- Практическая часть — 30 мин  
  - Загрузка и анализ реальных данных (15 мин)  
  - Создание графиков схемного контроля (15 мин)  
- Закрепление — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

- **Организационный момент**  
  Поприветствовать студентов, кратко огласить программу и цели занятия.  

- **Актуализация знаний**  
  Быстрый опрос: какие библиотеки Python уже использовали для работы с данными? Какие типы технических документов встречались?  

- **Введение в структуру технической документации**  
  Рассмотреть основные разделы спецификаций, схем и таблиц, показать примеры файлов (PDF, CSV).  

- **Работа с таблицами в Python (Pandas)**  
  Демонстрация чтения CSV, фильтрации, подсчёта статистики, преобразования типов.  

- **Визуализация схемного контроля (Matplotlib)**  
  Показать как строить линейные графики, графики ошибок, сравнение по времени.  

- **Загрузка и анализ реальных данных**  
  Студенты загружают пример CSV, проводят первичный анализ, определяют ключевые показатели.  

- **Создание графиков схемного контроля**  
  Написание кода, генерирующего графики, добавление подписей и легенд, сохранение изображений.  

- **Закрепление**  
  Быстрый контрольный тест из 5 вопросов по теме.  

- **Итог занятия**  
  Подведение итогов, ответы на вопросы, разъяснение домашнего задания.  

## Оборудование и материалы  

- Компьютеры с установленной IDE (VS Code, PyCharm) и Python 3.10+  
- Библиотеки: Pandas, NumPy, Matplotlib  
- Проектор/дисплей для демонстрации  
- Пример технической документации (CSV и PDF)  
- Учебник по Python и схемному контролю (стр. 45‑67)  

## Контроль и оценивание  

- Короткий тест (5 вопросов) в конце занятия  
- Оценка практической работы по критериям: корректность кода, правильность визуализации, комментарии в коде  
- Участие в дискуссии (обсуждение результатов)  

## Домашнее задание  

1. Найти в сети техническую спецификацию (PDF) любого промышленного устройства, экспортировать таблицы в CSV и написать скрипт, который выводит средние значения ключевых параметров.  
2. Создать график зависимости одного из параметров от времени, добавить подписи и легенду, сохранить как PNG.
