# Лабораторная работа № 1: Определение основных технических характеристик мобильных устройств

## Цель занятия  
1. Научить студентов применять Python для сбора и анализа технических характеристик мобильных устройств.  
2. Развить навыки работы с API и обработки JSON‑данных в реальных задачах.

## Задачи занятия  
- Ознакомить с источниками данных о мобильных устройствах (API, публичные датасеты).  
- Научить использовать библиотеки `requests`, `pandas` и `json`.  
- Практически применить методы фильтрации, сортировки и визуализации собранных данных.  
- Оценить качество полученных результатов и способность интерпретировать их.

## План занятия (90 минут)

- Организационный момент — 10 мин  
- Актуализация знаний — 15 мин  
- Изучение нового материала — 35 мин  
- Закрепление — 20 мин  
- Итог занятия — 10 мин  

## Содержание этапов  

### Организационный момент  
Краткое приветствие, проверка подключения к Интернету и наличия необходимых библиотек. Уточнение расписания лабораторной работы и формата взаимодействия.

### Актуализация знаний  
Обсуждение предыдущих занятий по работе с API и обработке JSON. Быстрый опрос студентов о том, какие типы данных они уже умеют обрабатывать в Python.

### Изучение нового материала  
1. Введение в открытый API (например, `https://api.phonespecs.com/`).  
2. Практика запросов с помощью `requests.get` и получение данных в формате JSON.  
3. Перевод JSON в `pandas.DataFrame`, базовые операции по очистке и фильтрации.  
4. Простой график (Matplotlib/Seaborn) по количеству устройств по разным параметрам (RAM, процессор, камера).  
5. Сохранение результатов в CSV и обсуждение структуры данных.

### Закрепление  
Пошаговое выполнение лабораторной задачи: студентам выдаётся набор вопросов, требующих получения конкретных характеристик (например, найти 5 устройств с наибольшим объёмом памяти и вывести их модели). Проверка кода и результатов, обсуждение ошибок и оптимизаций.

### Итог занятия  
Обсуждение проделанной работы, ответы на вопросы студентов, краткий обзор того, что было изучено и как применить навыки в следующих лабораторных работах.

## Оборудование и материалы  
- ПК со стабильным интернет‑соединением.  
- Python 3.10+ (установленные пакеты: `requests`, `pandas`, `matplotlib`, `seaborn`).  
- IDE (PyCharm, VS Code, Jupyter Notebook).  
- Проектор/экран для демонстрации примеров.  
- Учебник/слайды по работе с API и JSON.  

## Контроль и оценивание  
- Короткие вопросы по теории (2‑3 вопроса) в начале занятия.  
- Проверка выполнения лабораторной задачи: корректность запросов, правильность фильтрации и визуализации.  
- Оценка кода по критериям чистоты, эффективности и комментариев.  

## Домашнее задание  
1. Написать скрипт, который собирает данные о 10 устройствах с наибольшим количеством камер и сохраняет их в JSON‑файл.  
2. Подготовить краткий отчёт (максимум 2 стр. с) с выводами о тенденциях в развитии характеристик мобильных устройств за последние 5 лет.
