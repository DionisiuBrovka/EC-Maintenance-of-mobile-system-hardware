# Лабораторная работа № 2  
Использование специального оборудования для эксплуатационного обслуживания мобильных устройств  

## Цель занятия  
1. Научить студентов применять специализированное оборудование для диагностики и обслуживания мобильных устройств, используя Python для автоматизации процессов.  
2. Развить навыки работы с API и библиотеками, взаимодействующими с внешним оборудованием, а также умение анализировать полученные данные.

## Задачи занятия  
1. Ознакомиться с типами оборудования, применяемого в эксплуатации мобильных устройств.  
2. Научиться подключать оборудование к компьютеру и получать данные через Python.  
3. Реализовать простую программу для сбора и визуализации информации о состоянии устройства.  
4. Оценить эффективность работы оборудования и предложить рекомендации по обслуживанию.

## План занятия  

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 25 мин  
- Практическая работа — 25 мин  
- Закрепление — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

- **Организационный момент**  
  Приветствие, проверка присутствия, постановка целей и раздача материалов.  

- **Актуализация знаний**  
  Быстрый опрос о предыдущем опыте работы с оборудованием и Python, обсуждение ожиданий от лабораторной работы.  

- **Изучение нового материала**  
  Лектор объясняет принципы работы выбранного оборудования, демонстрирует подключение через USB/serial, а также вводит основные библиотеки (`pyserial`, `requests`, `pandas`).  

- **Практическая работа**  
  Студенты в группах подключают оборудование к ПК, запускают скрипт, который читает данные о батарее, температуре и подключаемой памяти, сохраняет их в CSV и строит график.  

- **Закрепление**  
  Быстрый тест с 5 вопросами на выбор, проверка кода на наличие ошибок, обсуждение результатов работы.  

- **Итог занятия**  
  Подведение итогов, раздача чек-листа по эксплуатации оборудования, ответы на вопросы.  

## Оборудование и материалы  

- Компьютеры с установленной IDE (PyCharm, VS Code или Jupyter Notebook)  
- Специальное оборудование: тестер зарядки/отключения, адаптеры USB, OTG‑кабели  
- USB‑кабели и переходники для подключения к ПК  
- Проектор/дисплей для демонстрации  
- Учебник «Инструментальное программное обеспечение (изучение Python)» – раздел «Работа с внешним оборудованием»  
- Скрипт‑шаблон, предоставленный преподавателем  

## Контроль и оценивание  

- Устный контроль: ответы на вопросы в ходе занятия  
- Тест: 5 вопросов с множественным выбором по материалу  
- Практическое задание: оценка работы скрипта (корректность данных, читаемость кода, визуализация)  

## Домашнее задание  

1. Расширить скрипт, добавив запись логов в файл `log.txt` с отметкой времени.  
2. Подготовить краткий отчёт (не более 2 страниц) о работе оборудования: описание проблемы, полученные данные, выводы и рекомендации.
