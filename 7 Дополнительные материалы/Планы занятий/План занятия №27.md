# Лабораторная работа № 6: Тестирование памяти с помощью программных методов контроля  

## Цель занятия  
1. Научить студентов применять программные методы контроля для измерения и анализа использования памяти в Python‑приложениях.  
2. Познакомить с инструментами `timeit`, `memory_profiler`, `tracemalloc` и их интеграцией в рабочий процесс разработки.  

## Задачи занятия  
- Описать основные понятия измерения потребления памяти.  
- Показать, как подключать и использовать профайлеры памяти в IDE.  
- Практически выполнить тесты по измерению памяти для заданных алгоритмов.  
- Научиться интерпретировать отчёты и принимать решения о рефакторинге.  

## План занятия  
- Организационный момент — 10 мин  
- Актуализация знаний — 15 мин  
- Изучение нового материала — 30 мин  
- Практическая часть – работа с профайлерами — 45 мин  
- Закрепление и обсуждение результатов — 20 мин  
- Итог занятия — 10 мин  

## Содержание этапов  

### Организационный момент  
Объявление темы, проверка присутствия, краткая презентация целей и структуры лабораторной работы.  

### Актуализация знаний  
Краткое повторение принципов работы Python‑памяти, динамического распределения и GC, а также обзор предыдущих лабораторных работ.  

### Изучение нового материала  
Введение в инструменты `timeit`, `memory_profiler` и `tracemalloc`. Демонстрация синтаксиса и основных параметров. Обсуждение различий между измерением времени и памяти, а также методов их сочетания.  

### Практическая часть – работа с профайлерами  
Студенты по группам выполняют задачи:  
1. Измеряют память, занимаемую функцией сортировки списка.  
2. Сравнивают разные реализации (через списки, генераторы, массивы NumPy).  
3. Профилируют программу, использующую большие объёмы данных, и выявляют утечки памяти.  

### Закрепление и обсуждение результатов  
Обсуждение выводов из профайлеров, оценка эффективности разных подходов, обмен опытом по оптимизации кода.  

### Итог занятия  
Подведение итогов, ответы на вопросы, раздача чек‑листа по тестированию памяти.  

## Оборудование и материалы  
- Студенты: ноутбук (Windows/Linux/macOS) с установленной Python 3.10+  
- IDE: PyCharm, VS Code или любой другой с поддержкой Python  
- Проектор/дисплей для демонстрации  
- Библиотеки: `timeit`, `memory_profiler`, `tracemalloc` (установить через pip)  
- Учебный материал: ссылки на официальную документацию, примеры кода  

## Контроль и оценивание  
- Быстрый устный опрос по ключевым понятиям (5 мин).  
- Проверка результатов профилирования в виде короткого отчёта (таблица с потреблением памяти и временем).  
- Оценка выполнения практических заданий по критериям: корректность кода, точность измерений, обоснование выбора решения.  

## Домашнее задание  
1. Написать программу, которая генерирует 1 млн случайных чисел, сохраняет их в список, а затем удаляет список. Сравнить потребление памяти до и после удаления с помощью `tracemalloc`.  
2. Составить краткий отчёт (не более 500 слов) о том, какие изменения в коде привели к снижению потребления памяти и почему.
